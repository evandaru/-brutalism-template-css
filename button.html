Buttons Showcase

Berbagai variasi tombol dengan kelas .btn dan modifier-nya.

1. Tombol Dasar

Interaksi fundamental. Termasuk link, button, dan state disabled.

Link as Button

2. Tombol Berwarna (Contextual)

Gunakan modifier .btn--[color] untuk memberikan konteks.

3. Tombol Terisi (Filled)

Tambahkan .btn--filled untuk penekanan visual yang lebih kuat.

4. Tombol dengan Ikon

Menggabungkan ikon SVG dan teks untuk kejelasan.

5. Tombol Efek & Blok

Gunakan .text-glitch-hover untuk efek unik dan .btn-block untuk lebar penuh.

6. Grup Tombol (Toggle)

Gunakan JavaScript untuk menambahkan kelas .is-active.
